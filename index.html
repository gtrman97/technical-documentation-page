Introduction

Python is an interpreted, high-level and general-purpose programming language. Python's design philosophy emphasizes code readability with its notable use of significant indentation.

Variables

Variables are containers for storing data values.

There are three main types of variables in Python: Integers (numbers), floats (decimal point numbers), and strings (words).

x = 99 y = 3.7 z = "Andrew" print(x) // output: 99 print(y) // output: 3.7 print(z) // output: Andrew
Lists

Lists are used to store multiple items in a single variable.

activities = ["gym", "guitar", "code"] print(activities[0]) // ouput: gym print(activities[1]) // output: guitar print(activities[2]) // output: code
If Statements

Conditions can be used in several ways, most commonly in "if statements"

if (x < 5): print("your number is less than 5") elif (x > 5): print("you number is greater than 5") else (x == 5): print("you number is 5")
For Loops

A for loop is used for iterating over a sequence (that is either a list, a tuple, a dictionary, a set, or a string).

nums = [1, 2, 3, 4, 5] for num in nums: print(nums) // Expected Output: // 1 // 2 // 3 // 4 // 5
Functions

A function is a block of code which only runs when it is called. You can pass data, known as parameters, into a function. A function can return data as a result.

x = 5; y = 3; def add(a, b): return a + b sum = add(x, y) print(sum) // output: 8
